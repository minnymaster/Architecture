# Лабораторная работа №1
**Тема:** Формулирование требований к программной системе
**Цель работы:** Научиться анализировать поставленную задачу, формулировать функциональные и нефункциональные требования к проектируемой системе.

---

## 1. Перечень стейкхолдеров

### Директор компании
  - Повышение квалификации сотрудников и соответствие их компетенций стратегическим целям компании.
  - Оптимизация процессов аттестации и обучения для снижения затрат на внешние тренинги.
  - Получение аналитических данных о уровне компетенций сотрудников для принятия управленческих решений.
  - Контроль за эффективностью обучения и аттестации через автоматизированные отчёты.

### HR-специалист
  - Автоматизация процесса оценки компетенций сотрудников.
  - Возможность гибко настраивать тесты и критерии оценки под задачи компании.
  - Получение детализированных отчётов по результатам аттестации для анализа и планирования обучения.
  - Снижение рутинной работы по обработке результатов тестирования.

### Сотрудник компании
  - Возможность пройти аттестацию в удобное время и в комфортном темпе.
  - Получение обратной связи по результатам тестирования для саморазвития.
  - Доступ к обучающим материалам и рекомендациям по улучшению компетенций.
  - Прозрачность процесса аттестации и объективность оценки.

---

## 2. Перечень функциональных требований

### Для директора компании
-  Просмотр сводных отчётов по компетенциям всех сотрудников в реальном времени.
-  Возможность установки целевых показателей компетенций для подразделений и сотрудников.
-  Получение уведомлений о критических изменениях в уровне компетенций.
-  Экспорт отчётов в форматах PDF и Excel для презентаций и анализа.

### Для HR-специалиста
-  Создание и редактирование тестов для оценки компетенций,включая добавление вопросов, установку времени прохождения, настройку баллов.
-  Автоматическая генерация тестов на основе компетенций с использованием LLM-модели.
-  Назначение тестов конкретным сотрудникам или группам сотрудников.
-  Мониторинг процесса прохождения тестов (статус, время, результаты).
-  Формирование индивидуальных и групповых отчётов по результатам аттестации.
-  Настройка уведомлений для сотрудников о предстоящих аттестациях.
-  Возможность добавления обучающих материалов и рекомендаций для сотрудников на основе результатов тестирования.

### Для сотрудника компании
-  Регистрация и аутентификация в системе с использованием корпоративных учётных данных.
-  Просмотр списка назначенных тестов и сроков их прохождения.
-  Просмотр результатов тестирования и сравнение с предыдущими попытками.
-  Получение персонализированных рекомендаций по улучшению компетенций.
-  Доступ к обучающим материалам и тренировочным тестам.
-  Уведомления о новых тестах, результатах и рекомендациях.
